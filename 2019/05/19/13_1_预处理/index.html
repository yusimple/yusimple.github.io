13.1 预处理

Course: C_learning
Last Edited: May 17, 2019 1:54 PM
章节: 13.1

  1. 预处理的概念

    预处理是指在编译之前对源程序所做的处理称为预处理，指导编译器对源代码进行操作

  2. 指示与语句的区别

    • 指示一般以‘#’号开头，是给编译器的命令(并不产生二进制程序)

      #define PI 3.14
#include<stdio.h>

    • 语句以分号为结尾，被编译器翻译为二进制程序

      int i;//声明性语句
y=x+1;//指令性语句

  3. 预处理1——文件包含

    • 文件包含两种形式

      #include<stdio.h>//.C语言自身库的头文件，搜索系统头文件所在的目录
#include"myheader.h"//.C语言搜索当前目录，在搜索系统头文件目录

  4. 预处理2——条件编译

    • 如果一个头文件被包含多次，可能会产生编译错误，此时需要条件编译语句

      #ifndef MY_HEADER_H
#define MY_HEADER_H
#endif//栗子之一

#define DEBUG 1//这里是总开关
#if DEBUG
        printf("");
        printf("");
#endif//栗子之二
//总之一句话带#号的是给编译器看的
# if defined(MAX)
......
#elif define(LINUX)
......
#else
#error No operationg system specified
#endif//栗子之三
